# 🛒 StoRê Sport - Landing Page Premium

Uma landing page de alta performance desenvolvida para a **StoRê Sport**, focada em apresentar a curadoria de vestuário fitness com tecnologia Rala Bela. O projeto une design minimalista "dark mode" com uma experiência de usuário fluida e responsiva.

## 🚀 Demonstração
O projeto está publicado e pode ser acessado em: 
[https://nunes2705.github.io/store-sport-premium/](https://nunes2705.github.io/store-sport-premium/)

## 🛠️ Tecnologias Utilizadas
Este projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno e ferramentas de IA:

- **HTML5 & CSS3**: Estrutura e estilização customizada.
- **Tailwind CSS**: Framework utilitário para um design responsivo e moderno.
- **JavaScript**: Lógica para interatividade e componentes dinâmicos.
- **GitHub Pages**: Hospedagem e deploy contínuo.
- **FontAwesome**: Iconografia profissional.
- **IA Generativa**: Utilizada para otimização de código e estruturação de dados.

## 📋 Funcionalidades
- [x] **Design Responsivo**: Adaptável para dispositivos móveis, tablets e desktop.
- [x] **Vitrine Dinâmica**: Exibição da coleção Outono/Inverno com imagens reais de produtos.
- [x] **Seção de Tecnologia**: Destaque para os diferenciais têxteis da marca.
- [x] **Integração com Canais de Venda**: Botão direto para WhatsApp e formulário de feedback via FormSubmit.
- [x] **Otimização de Performance**: Carregamento rápido e estrutura de código limpa.
